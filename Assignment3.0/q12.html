HTML Entity Demonstration

Reserved Characters

To display the less-than sign (<) in HTML, you must write <

To display the greater-than sign (>) in HTML, you must write >

To display an ampersand (&), you must write &

To display a double-quote ("), you can write "

To display a single-quote ('), you can write '

Other Common Entities

The copyright symbol is © (2025).

The trademark symbol is ™.

The registered trademark symbol is ®.

Example in a sentence:

The HTML code for a paragraph is <p>This is a "test"</p>. This content is © MyBrand™.
