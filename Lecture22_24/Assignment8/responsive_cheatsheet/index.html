Filters

Applies graphical effects like blur or color shifting. filter: grayscale(100%);

Media Queries

Used to apply styles based on viewport characteristics. @media (max-width: 600px) { ... }

Responsive Design

An approach to make web pages render well on a variety of devices.

Typography

Use relative units like rem, em, and vw for scalable text. font-size: clamp(1rem, 2vw, 1.5rem);

Flexbox

A 1D layout model for distributing space. display: flex; justify-content: center;

CSS Grid

A 2D layout model for rows and columns. display: grid; grid-template-columns: 1fr 1fr;
