Box-Sizing Comparison

box-sizing: content-box (Default)

I am set to 300px width.

Actual Width: 364px
(300px width + 30px padding-left + 30px padding-right + 2px border-left + 2px border-right)

The padding and border are added outside the defined width, making the box larger.

box-sizing: border-box

I am set to 300px width.

Actual Width: 300px
(Width includes padding and border)

The padding and border are included inside the defined 300px width, making layout predictable.
